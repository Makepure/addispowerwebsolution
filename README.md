# Addis Power AI & Web Solution

A professional website for Addis Power AI & Web Solution, featuring AI chatbot integration and pricing plans in Ethiopian Birr.

## Features

- **Responsive Design** - Works on all devices
- **AI Chatbot** - Interactive assistant for customer support
- **Pricing Plans** - Clear pricing in ETB with three tiers
- **Service Pages** - Detailed service information
- **Contact Form** - Easy customer inquiry system
- **Modern UI** - Clean and professional design
